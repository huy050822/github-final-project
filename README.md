# Simple Interest Calculator

This is a simple web application that calculates simple interest based on the principal amount, annual rate of interest, and the time period.

## Features
- Input principal amount.
- Input annual rate of interest.
- Input time in years.
- Calculate and display the simple interest.

## How to use
1. Open the application in your browser.
2. Enter the required values in the input fields.
3. Click the "Calculate" button to see the result.
